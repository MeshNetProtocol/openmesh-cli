## 3. 实现内容

### 3.1 流量采集服务

**文件**：`nodes/{node-a,node-b,node-c}/traffic-collector.py`

**核心功能**：
```python
def simulate_traffic(user_id):
    """模拟用户流量增量"""
    upload_delta = random.randint(0, 102400)    # 0-100KB
    download_delta = random.randint(0, 512000)  # 0-500KB
    return upload_delta, download_delta

def report_traffic(node_id, user_id, upload_bytes, download_bytes):
    """上报流量到记账服务"""
    url = f"{METERING_SERVICE_URL}/api/v1/metering/report"
    payload = {
        "node_id": node_id,
        "user_id": user_id,
        "upload_bytes": upload_bytes,
        "download_bytes": download_bytes
    }
    response = requests.post(url, json=payload, timeout=5)
    return response.json()
```

**运行逻辑**：
1. 每 10 秒执行一次采集循环
2. 为每个用户（alice/bob/charlie）模拟流量增量
3. 上报到记账服务
4. 接收流量桶状态响应
5. 如果流量桶耗尽，记录警告
