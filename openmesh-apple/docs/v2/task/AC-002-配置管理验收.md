## 验收目标

验证配置管理系统是否满足生产环境要求,包括:
1. 配置加载功能是否正常
2. 环境变量覆盖是否工作
3. 配置验证是否有效
4. 安全性是否达标

## 功能测试

### 测试 1: 配置文件加载

**测试步骤**:
```bash
cd openmesh-apple/metering-service

# 创建测试配置文件
cat > config.test.yaml <<EOF
database:
  host: testhost
  port: 5433
  name: testdb
  user: testuser
  password: testpass
  ssl_mode: disable

collector:
  interval: 20s
  timeout: 10s

server:
  port: 9090
  log_level: debug
EOF

# 测试加载配置
go run cmd/metering/main.go --config config.test.yaml --dry-run
```

**预期结果**:
- [ ] 配置文件成功加载
- [ ] 所有配置项正确解析
- [ ] 输出显示正确的配置值

### 测试 2: 环境变量覆盖

**测试步骤**:
```bash
# 设置环境变量
export METERING_DATABASE_HOST=envhost
export METERING_DATABASE_PORT=5434
export METERING_SERVER_PORT=8888

# 加载配置(环境变量应覆盖配置文件)
go run cmd/metering/main.go --config config.test.yaml --dry-run
```

**预期结果**:
- [ ] 环境变量成功覆盖配置文件
- [ ] database.host = "envhost"
- [ ] database.port = 5434
- [ ] server.port = 8888

### 测试 3: 默认值

**测试步骤**:
```bash
# 创建最小配置文件
cat > config.minimal.yaml <<EOF
database:
  host: localhost
  port: 5432
  name: metering
  user: user
  password: pass
EOF

# 加载配置
go run cmd/metering/main.go --config config.minimal.yaml --dry-run
```

**预期结果**:
- [ ] 未指定的配置项使用默认值
- [ ] collector.interval = 15s (默认值)
- [ ] server.port = 8090 (默认值)
- [ ] logging.level = "info" (默认值)

### 测试 4: 配置验证

**测试步骤**:
```bash
# 测试无效端口
cat > config.invalid.yaml <<EOF
database:
  host: localhost
  port: 99999  # 无效端口
  name: metering
  user: user
  password: pass
EOF

go run cmd/metering/main.go --config config.invalid.yaml 2>&1 | grep -i "validation"

# 测试缺少必需字段
cat > config.missing.yaml <<EOF
database:
  host: localhost
  # 缺少 port, name, user, password
EOF

go run cmd/metering/main.go --config config.missing.yaml 2>&1 | grep -i "required"
```

**预期结果**:
- [ ] 无效配置被拒绝
- [ ] 错误信息清晰指出问题
- [ ] 程序不会启动

### 测试 5: 敏感信息处理

**测试步骤**:
```bash
# 检查配置文件示例
grep -i "password" configs/config.example.yaml
grep -i "secret" configs/config.example.yaml

# 确认使用环境变量占位符
cat configs/config.example.yaml | grep '\${.*}'
```

**预期结果**:
- [ ] 示例配置不包含真实密码
- [ ] 使用 ${VAR} 占位符
- [ ] .env.example 不包含真实密钥
