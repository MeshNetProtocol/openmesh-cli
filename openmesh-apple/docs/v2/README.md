## 关键技术决策

- **协议选择**: Hysteria2（支持用户级流量统计、多用户认证、高性能）
- **架构模式**: 4 层架构（Discovery、Client、Data、Provider Plane）
- **流量计费**: 中心化扣减 + 定期拉取（Phase 0-1）
- **供应商切换**: 动态配置更新 + 无感切换
