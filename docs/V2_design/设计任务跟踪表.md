## 使用说明

### 如何更新此表格

1. **任务状态变更**：
   - 修改"设计任务清单"表格中的"状态"列
   - 更新"开始日期"或"完成日期"
   - 在"更新日志"中记录变更

2. **任务完成**：
   - 将状态改为 ✅ 已完成
   - 填写完成日期
   - 在"已完成任务详情"章节添加任务总结
   - 更新"设计阶段总体进度"

3. **遇到阻塞**：
   - 在"当前阻塞项"表格中添加记录
   - 在任务的"备注"列说明阻塞原因

4. **发现风险**：
   - 在"风险与问题"表格中添加记录
   - 制定缓解措施

### 审核流程

每个设计文档完成后需要经过以下审核：

1. **自检**：负责人完成后自行检查完整性和一致性
2. **项目负责人审核**：检查是否符合需求和项目目标
3. **团队评审**：关键设计文档需要团队评审
4. **标记完成**：通过审核后在本表格中标记为"已完成"
