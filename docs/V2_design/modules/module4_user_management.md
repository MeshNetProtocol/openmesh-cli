# 模块 4: 用户管理模块

## 模块状态

**当前状态**: ⏳ 待开发  
**最后更新**: 2026-04-06

## 1. 模块概述

本模块负责动态管理 Xray 服务器的用户,支持实时添加、删除用户,无需重启服务。

### 1.1 核心功能

- **动态添加用户**: 使用 xtlsapi 动态添加用户到 Xray
- **动态删除用户**: 从 Xray 删除用户
- **用户查询**: 查询 Xray 中的用户列表
- **批量操作**: 支持批量添加/删除用户
- **多服务器管理**: 同时管理多个 Xray 服务器的用户

## 2. 技术方案

### 2.1 核心技术选型

| 技术组件 | 选型 | 用途 |
|---------|------|------|
| 用户管理库 | xtlsapi (Python) | 调用 Xray gRPC API |
| gRPC 客户端 | Go grpc | 直接调用 Xray API |

### 2.2 动态用户管理

**问题**: Xray 的 `xray api adu` 命令行工具不工作

**解决方案**: 使用 xtlsapi 库直接调用 Xray 的 gRPC API

**实现方案**:
- 添加用户: `XrayClient.add_client()` → gRPC `HandlerService.AlterInbound`
- 删除用户: `xray api rmu` 命令 → gRPC `HandlerService.AlterInbound`
- 查询用户: `xray api inbounduser` 命令 → gRPC `HandlerService.GetInboundUsers`
