# 基于区块链地址的订阅式 VPN 服务 - 项目总览

## 1. 项目概述

本项目旨在构建一个基于区块链地址认证的订阅式 VPN 服务系统。用户通过区块链支付订阅费用后，可以使用客户端访问分布在全球不同地区的 Xray 服务器进行 VPN 转发。系统支持动态用户管理、多服务器流量汇总、自动续费等功能。

## 2. 核心特性

- **区块链地址认证**: 使用区块链地址替代传统的 email 作为用户标识
- **订阅制付费**: 用户支付订阅费后自动获得访问权限
- **多服务器支持**: 用户可以连接到不同地区的多个 Xray 服务器
- **流量统计与汇总**: 实时统计并汇总用户在所有服务器的流量使用情况
- **流量持久化**: 防止服务器重启后流量数据丢失
- **动态用户管理**: 实时启用/禁用用户，无需重启服务
- **自动续费与停服**: 订阅到期自动扣款，扣款失败自动停止服务
- **WebSocket 实时推送**: 实时推送流量更新到客户端

## 7. 技术栈

### 后端
- **Go**: HTTP 订阅服务、WebSocket 服务
- **Python**: xtlsapi 用户管理脚本
- **Xray**: VPN 流量转发服务器

### 前端
- **HTML/CSS/JavaScript**: Web 管理界面
- **WebSocket**: 实时流量推送

### 客户端
- **Sing-box**: VPN 客户端

### 存储
- **JSON 文件**: 用户配置和流量数据持久化
- **PostgreSQL/MySQL**: 订阅记录和支付记录（计划中）

### 区块链
- **以太坊/Polygon**: 支付和订阅管理（计划中）
